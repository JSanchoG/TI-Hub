Konfiguracja interfejsów sieciowych w systemie Linux Ubuntu 18.04

Aby zacząć konfigurację interfejsu sieciowego, w wierszu poleceń należy wpisać ifconfig, jeżeli wystąpi błąd, konieczna będzie instalacja narzędzi sieciowych za pomocą polecenia sudo apt install net-tools, pamiętając o przedrostku sudo, dzięki któremu polecenie wykonywane będzie z uprawnieniami administratora

Następnie należy wykonać polecenie ifconfig, wyświetlą się interfejsy sieciowe
potem trzeba przejść do /etc/netplan używając polecenia cd

Następnie należy edytować plik 01-network-manager-all.yaml posługując się poleceniem nano z przedrostkiem sudo (jeśli polecenie zostanie wykonane bez przedrostka sudo, nie będzie można zapisać edytowanego pliku)
sudo nano 01-network-manager-all.yaml

Można tu zmienić adres IP danej karty sieciowej, dodać bramę lub adresy DNS, pamiętając o wcięciach wynoszących dwie spacje

Aby zapisać edytowany plik, należy wciśnąć Ctrl+O, a następnie bez zmiany nazwy pliku wcisnąć Enter

Zapisać zmiany można wykonując polecenie sudo netplan apply

Aby skonfigurować drugi interfejs sieciowy, należy trzy razy się cofnąć, używając polecenia cd, a następnie edytować plik /etc/network/interfaces za pomocą polecenia nano z przedrostkiem sudo
sudo nano /etc/network/interfaces

Zresetować kartę sieciową można poleceniem
sudo /etc/init.d/networking restart
lub restartując system
